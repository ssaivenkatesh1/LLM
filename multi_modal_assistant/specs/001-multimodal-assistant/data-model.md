# Data Model: Multimodal Assistant

**Date**: 2025-11-07
**Plan**: [plan.md](plan.md)

This document defines the key data entities for the Multimodal Assistant feature, based on the feature specification.

## 1. Query

Represents a single user submission to the assistant. This entity is transient and is not persisted.

| Field | Type | Description | Validation Rules |
| :--- | :--- | :--- | :--- |
| `text` | String (optional) | The text question asked by the user. | Max length: 2000 characters. |
| `image` | File (optional) | The image file uploaded by the user. | Must be one of: PNG, JPG, GIF. Max size: 5MB. |

**State Transitions**: A `Query` object is created on submission and processed by the backend. It is discarded after the `Answer` is generated.

## 2. Answer

Represents the response generated by the assistant for a given `Query`. This entity is transient and is not persisted.

| Field | Type | Description |
| :--- | :--- | :--- |
| `text` | String | The textual answer generated by the AI model. |
